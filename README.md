# S-Day · 极速外科病人管理

> **手术日 (Surgery Day)** — 专为外科医生设计的极简患者管理工具  
> 当前版本：`v0.1.0-beta` · 内测中

---

## 简介

S-Day 是一款专为外科医生在手术日快速管理患者信息的 iOS 原生应用。

在高强度的临床工作中，医生往往需要在极短的时间内快速定位、更新、追踪大量患者的手术状态。S-Day 的设计哲学只有一个：**极速**。没有繁琐的表格，没有多余的点击，只需一行文字输入，即可完成患者建档与标签分类。

---

## 核心功能

### 📋 患者管理
- **快速建档**：在输入框直接键入患者信息，支持 `#标签` 语法同步打标
- **术前 / 术后分组**：根据手术日期自动将患者归类至对应列表
- **按日期折叠**：同一手术日的患者汇聚为可折叠的分组，一目了然
- **左滑快捷操作**：侧滑患者条目即可快速设置手术日期、添加标签或删除

### 🏷️ 标签系统
- `#需追踪`、`#收藏` 等内置系统标签，自动高亮显示
- 支持完全自定义标签颜色，统一管理全局标签库
- 输入框内输入 `#关键词` 即自动补全已有标签

### 🔍 全局与局部搜索
- **术前 / 术后页**：实时过滤当前列表，输入即生效，无需跳转
- **概览页**：全局搜索跨越所有患者，自动标注术前 / 术后归属

### 📅 智能日期标题
- 今天手术 / 明天手术 / 昨天手术等相对日期自动识别
- 本年度日期自动省略年份，减少视觉干扰
- 标题同时显示星期几，方便排期规划

### 🔐 隐私保护
- 支持 **Face ID / Touch ID / 设备密码** 验证后方可进入应用
- App 切入后台时自动上锁，防止多任务截图导致信息泄露
- 隐私锁保护开关可随时在设置中开启或关闭

### 🎨 外观设置
- 支持跟随系统 / 强制浅色 / 强制深色三种主题模式

### 💾 数据导入 / 导出
- 一键将全部患者数据（包括标签、顺序、颜色配置、外观与隐私设置）导出为 JSON 文件
- 支持从文件 App 导入，完整恢复所有数据
- 导出文件自动命名（含时间戳）并存入「文件 · 我的 iPhone · S-Day」目录
