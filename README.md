# S-Day

> 专为外科医生设计的 iPhone 原生患者清单工具。  
> 当前版本：`v0.1.1-beta`

S-Day 的目标很明确：让医生在高密度手术日里，用尽可能少的操作完成患者录入、分组、检索、标记与导出。

它不是完整 HIS，也不是复杂表格系统，而是一套围绕“今天要看的人”设计的轻量工作流。

## 适合什么场景

- 术前患者快速录入与排期整理
- 术后患者继续追踪与标签筛查
- 门急诊、病房、手术室之间的临时清单管理
- 需要离线、本地、低学习成本的个人工作台

## 当前能力

### 患者与清单

- 术前、术后双列表分开管理
- 依据手术日期自动分组
- 支持组内拖动排序
- 分组支持折叠/展开
- 右滑患者条目可直接进入多选模式
- 左滑患者条目可快速设定日期、打标签或删除

### 极简录入

- 顶部常驻“新病人”输入行，适合连续录入
- 一行文字即可建档
- 输入 `#标签` 可自动抽取为标签胶囊
- 已有标签支持联想补全
- 支持回车快速提交

### 标签系统

- 内置系统标签：`需追踪`、`收藏`
- 自定义标签自动纳入全局标签库
- 标签支持改名、删除、颜色调整
- 标签颜色支持自动分配和手动覆盖
- 会记录最近使用标签，优化标签排序

### 批量操作

- 多选后可批量删除
- 多选后可批量设置手术日期
- 多选后可批量添加标签
- 多选后可一键复制导出文本
- 分组标题支持整组勾选

### 搜索与概览

- 术前页实时搜索
- 术后页实时搜索
- 概览页全局搜索全部患者
- 概览页显示术前/术后总数
- Tab 会对带有 `需追踪` 标签的患者显示 badge

### 隐私与数据

- 可启用 Face ID / Touch ID / 设备密码解锁
- App 回到前台时自动重新鉴权
- 支持完整 JSON 导出
- 支持 JSON 导入并覆盖恢复
- 导出会包含患者、标签颜色、外观设置和隐私锁设置
- 设置页支持一键清除全部数据

### 体验细节

- 智能日期标题：今天 / 明天 / 昨天 / 本年简写日期
- 自定义主题：跟随系统、浅色、深色
- 多选模式下列表布局稳定，避免内容跳动
- 成功操作带触觉反馈与轻量 toast

## 技术栈

| 项目 | 说明 |
| --- | --- |
| 平台 | iOS |
| 语言 | Swift |
| UI | SwiftUI |
| 持久化 | SwiftData |
| 生物认证 | LocalAuthentication |
| 文件导入导出 | UniformTypeIdentifiers + `fileImporter` / `fileExporter` |

## 系统要求

- iOS 26.2+
- Xcode 26.0+ 进行编译和部署

## 运行项目

1. 使用 Xcode 打开 `S-Day app.xcodeproj`
2. 选择自己的签名团队
3. 在模拟器或真机上运行

项目当前适合个人使用和小范围内测，默认数据全部保存在本地。
