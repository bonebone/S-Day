# 更新日志

记录本项目的重要功能变化与交互调整。

## [0.3.0] - 2026-04-05

### 新增

- 新增长按桌面图标快捷菜单，支持直接从 Home Screen 进入“添加新病人”
- 新增从快捷菜单进入后自动跳转术前页，并聚焦“新病人”输入框、自动弹出软键盘

### 调整

- 统一版本号到 `0.3.0`，作为当前阶段的功能里程碑版本
- 重写 README，按当前产品能力重新整理定位、功能、隐私和使用说明

## [0.2.0] - 2026-03-20

### 新增

- 新增导入数据确认界面，需要输入确认文字才能执行导入操作
- 新增危险操作统一的安全确认机制（导入和清除数据均需文字确认）

### 调整

- 统一导入和清除数据确认界面的尺寸和样式
- 改善取消按钮的视觉对比度，提高可识别性
- 导入确认界面从全屏改为底部弹出 sheet 形式，提升用户体验

### 修复

- 修复取消按钮在某些主题下不够明显的视觉问题

## [0.1.1-beta] - 2026-03-14

### 新增

- 新增多选模式下的批量删除、批量打标签、批量设日期、批量文本导出。
- 新增全局标签管理，支持系统标签、自定义标签、重命名、删除和颜色调整。
- 新增 JSON 导入导出，覆盖患者数据、标签颜色、主题设置和隐私锁状态。
- 新增 Face ID / Touch ID / 设备密码隐私锁。
- 新增概览页搜索与 `需追踪` 标签 badge 提示。

### 调整

- 优化病人输入体验，支持行内 `#标签` 抽取、自动补全和回车提交。
- 优化术前/术后日期分组标题，支持相对日期显示。
- 优化多选模式布局稳定性，进入和退出时列表不再明显跳动。
- 优化术前页新增病人的插入反馈：确认新增后，现有病人条目会随列表重排平滑下移。
- 重写项目 README，使文档与当前功能保持一致。

### 修复

- 修复进入多选模式时列表内容整体位移的问题。
- 修复多选模式下手术日期分组标题左侧留白过大的问题。

## [0.1.0-beta] - 2026-03-14

### 初始版本

- 提供术前/术后患者列表、本地存储、搜索、侧滑快捷操作和主题设置。
